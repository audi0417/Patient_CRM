系統已內建完整的測試資料，只需執行一個指令即可開始使用！

## 功能展示

初始化後，您可以立即體驗：

### 1. 患者管理
- 查看 8 位模擬患者的完整資料
- 按標籤或群組篩選患者
- 編輯患者資訊

### 2. 健康數據分析
- 切換到「健康數據分析」頁面
- 點擊「群組」視圖查看按群組分類的患者
- 查看每位患者的健康數據趨勢圖表

### 3. 群組管理
- 前往「系統設定」→「標籤與群組管理」
- 查看已建立的 4 個群組
- 管理群組成員

### 4. 標籤系統
- 使用標籤快速篩選和分類患者
- 在患者卡片上查看標籤

## 重置資料

如果需要重新開始，隨時可以重新執行：

```bash
npm run init-db
```

**注意**: 這會完全覆蓋現有資料！

## 資料位置

所有資料儲存在：

```
data/
├── users.json              # 使用者帳號
├── tags.json               # 標籤
├── groups.json             # 群組
├── patients.json           # 患者基本資料
├── bodyComposition.json    # 體組成記錄
└── vitalSigns.json         # 生命徵象記錄
```

## 常見問題

**Q: 我忘記密碼了怎麼辦？**
A: 重新執行 `npm run init-db` 即可重置為 admin / Admin123

**Q: 可以修改模擬資料嗎？**
A: 可以編輯 `scripts/initDatabase.js` 來自訂初始資料

**Q: 正式使用時需要清除模擬資料嗎？**
A: 是的，正式使用前建議清空 `data/` 目錄，手動建立真實資料

**Q: 資料會保存嗎？**
A: Web 版本使用 localStorage 保存，Electron 版本使用文件系統
