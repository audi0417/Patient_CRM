# 📍 預約類型顏色編碼系統

## 🎨 改進概述

使用**整個卡片背景顏色**來表示預約類型。每種預約類型都有獨特的顏色，使得：
- ✅ 快速識別預約類型
- ✅ 視覺效果清晰直觀

- ✅ 提高用戶辨識效率

## 🎨 實現代碼

### 顏色映射函數
```tsx
const getAppointmentTypeColors = (type: string) => {
  switch (type) {
    case "定期回診":
      return "bg-purple-100 text-purple-900 dark:bg-purple-950 dark:text-purple-100";
    case "追蹤檢查":
      return "bg-blue-100 text-blue-900 dark:bg-blue-950 dark:text-blue-100";
    case "健康檢查":
      return "bg-green-100 text-green-900 dark:bg-green-950 dark:text-green-100";
    case "復健治療":
      return "bg-orange-100 text-orange-900 dark:bg-orange-950 dark:text-orange-100";
    default:
      return "bg-gray-100 text-gray-900 dark:bg-gray-800 dark:text-gray-100";
  }
};
```

### 應用方式
```tsx
<div className={cn(
  "text-xs p-2.5 rounded font-medium",
  getAppointmentTypeColors(apt.type),  // 應用背景和文字顏色
  // ... 其他類別
)}>
  {/* 卡片內容 */}
</div>
```

## 🔄 與狀態的互動

預約的**狀態**（已預約/已完成/已取消）通過**文字透明度和樣式**表示：

```
已預約：   完整不透明文字
已完成：   半透明文字（opacity-75）
已取消：   刪除線 + 半透明文字（opacity-60）
```

這樣既能表示類型（背景顏色），也能表示狀態（文字樣式）。
