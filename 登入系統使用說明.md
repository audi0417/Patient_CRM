# 患者管理系統 - 登入系統使用說明

## 🚀 快速開始

### 登入系統

1. **開啟應用程式**
   - 訪問 http://localhost:8080/
   - 進入登入頁面

2. **輸入帳號密碼**
   - 使用管理員提供的帳號資訊登入
   - 密碼要求:
     - 至少8字元
     - 含大小寫字母及數字

3. **開始使用**
   - 登入成功後即可開始管理病患資料

**注意**: Super Admin 帳號需透過後台系統建立,一般使用者帳號可由 Super Admin 或 Admin 在系統內建立

## 👥 使用者角色說明

### 超級管理員 (Super Admin)
- 擁有系統最高權限
- 可管理所有使用者帳號
- 可管理醫院設定
- 完整的病患資料存取權

### 管理員 (Admin)
- 可管理使用者帳號
- 完整的病患資料存取權
- 無法修改醫院設定

### 一般使用者 (User)
- 可查看、編輯病患資料
- 可管理預約
- 無法刪除病患
- 無法管理使用者

## 📋 常用功能

### 新增使用者 (需要 Admin 以上權限)

1. 點擊頂部導航列的「使用者管理」
2. 點擊「新增使用者」按鈕
3. 填寫使用者資訊並選擇角色
4. 點擊「建立」

### 編輯使用者

1. 在使用者列表中點擊「編輯」圖示
2. 修改使用者資訊
3. 點擊「儲存變更」

### 重設密碼

1. 在使用者列表中點擊「鑰匙」圖示
2. 輸入新密碼(需符合密碼規則)
3. 點擊「重設密碼」

### 登出

1. 點擊右上角的使用者名稱
2. 選擇「登出」

## 🔒 安全須知

### 密碼規則
- 至少 8 個字元
- 包含至少一個大寫字母
- 包含至少一個小寫字母
- 包含至少一個數字

### 會話管理
- Token 有效期為 24 小時
- 過期後需重新登入
- 建議每日使用完畢後登出

### 帳號安全
- 妥善保管您的密碼
- 不要與他人分享帳號
- 定期變更密碼(請聯絡管理員協助)

## ❓ 常見問題

**Q: 忘記密碼怎麼辦?**
A: 請聯絡系統管理員為您重設密碼。

**Q: 可以自己修改密碼嗎?**
A: 目前需要聯絡管理員協助變更密碼。

**Q: 為什麼我看不到「使用者管理」選項?**
A: 只有超級管理員和管理員可以看到此選項。

**Q: 刪除使用者會影響病患資料嗎?**
A: 不會,病患資料與使用者帳號是獨立的。

**Q: 可以有多個超級管理員嗎?**
A: 可以,超級管理員可以建立其他超級管理員帳號。

**Q: 系統會記錄使用者操作嗎?**
A: 目前版本會記錄最後登入時間,未來版本將加入更詳細的操作記錄。

## 💡 使用技巧

1. **快速切換功能**
   - 使用頂部導航列快速切換不同功能區

2. **使用者狀態管理**
   - 可以暫時停用使用者帳號而不刪除
   - 停用的使用者無法登入但資料保留

3. **角色選擇**
   - 根據實際需求分配適當的角色
   - 一般員工建議使用「一般使用者」角色
   - 需要管理權限的主管使用「管理員」角色
