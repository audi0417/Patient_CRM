# 患者管理系統 - 快速開始指南

## 🎯 系統已就緒

登入系統已完成整合,所有功能正常運作!

## 🚀 立即開始使用

### 方法一: 開發模式

開發伺服器已啟動:
- 網址: **http://localhost:8080/**
- 開啟瀏覽器訪問即可使用

如需重新啟動:
```bash
npm run dev
```

### 方法二: 生產模式

1. 編譯專案:
```bash
npm run build
```

2. 預覽生產版本:
```bash
npm run preview
```

3. 或啟動 Electron 版本:
```bash
npm run electron:dev
```

## 📝 使用步驟

1. **開啟應用程式**
   - 訪問 http://localhost:8080/
   - 進入登入頁面

2. **登入系統**
   - 使用管理員提供的帳號密碼登入
   - 開始管理病患資料

3. **新增其他使用者** (Super Admin / Admin 權限)
   - 點擊右上角使用者選單
   - 選擇「使用者管理」
   - 新增其他員工帳號

**注意**: Super Admin 帳號需透過後台系統建立

## 👥 使用者角色說明

### 超級管理員 (Super Admin)
- 完整系統權限
- 可管理所有使用者
- 可修改醫院設定

### 管理員 (Admin)
- 可管理使用者帳號
- 完整病患資料權限
- 無法修改醫院設定

### 一般使用者 (User)
- 可查看、編輯病患資料
- 可管理預約和健康記錄
- 無法刪除病患或管理使用者

## 🔒 安全提醒

1. **保管好您的密碼**
   - 不要與他人分享
   - 定期更換密碼

2. **使用完畢請登出**
   - 點擊右上角使用者選單
   - 選擇「登出」

3. **定期備份資料**
   - 在「設定」頁面進行資料備份

## ⚙️ 系統架構

- **前端**: React + TypeScript + Vite
- **UI**: shadcn/ui + Tailwind CSS
- **認證**: JWT + bcrypt
- **資料儲存**: localStorage (單機版)
- **桌面版**: Electron

## 🆘 常見問題

**Q: 如何重設密碼?**
A: 聯絡系統管理員為您重設。

**Q: 可以有多個管理員嗎?**
A: 可以,超級管理員可建立其他管理員。

**Q: 資料儲存在哪裡?**
A: 目前版本使用瀏覽器 localStorage,桌面版將使用 SQLite。

**Q: 如何備份資料?**
A: 在「設定」頁面可進行資料備份和匯出。
